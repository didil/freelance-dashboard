h1. Freelance Dashboard

A Ruby on Rails dashboard app for freelancers allowing to :

* Search for jobs on Elance & Odesk by keywords

This rails app makes use of :

* Twitter Bootstrap for the UI 
* Devise, Cancan and Rolify for Authentification / Authorization / Roles
* Rspec and Capybara / PhantomJS - Poltergeist for Integration and Unit testing
* New relic for Performance monitoring
* Heroku for cloud hosting
* Google Analytics for site analytics

You can see it live on http://freelance-dashboard.herokuapp.com
