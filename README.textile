h1. Freelance Dashboard

A Ruby on Rails dashboard app for freelancers allowing to :

* Search for jobs on Elance
* Search for jobs on Odesk

Integration and Unit testing with Rspec and Capybara / Poltergeist

Live on http://freelance-dashboard.herokuapp.com
